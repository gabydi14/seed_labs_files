These mobile security SEED labs (called MobiSEED labs) need a different set of virtual machines. Please download the following VMs and user manuals for the labs hosted on this page.

Android VM (Android 5.1)

  • SEEDAndroid5.1.zip. This VM has two operating systems installed, one is the Android OS (version 5.1), and the other is a recovery OS (a reduce-sized Ubuntu). The recovery OS is only needed in the Device Rooting Lab.
    • The MD5 checksum of the file: 9ea25ea59d00311aa15d86dc576e6eb8.
    • User Manual for the Android OS
    • User Manual for the Recovery OS
    • How to Run SEEDAndroid on VirtualBox

Ubuntu VM (Ubuntu 14.04)

  • MobiSEEDUbuntu14.04_x64.zip: This Ubuntu VM is used to conduct development-related tasks, such as building app and native code, adb to Android VM, reverse engineering apps, etc. We have installed various tools and Android SDK/NDK that are needed for MobiSEED labs.
    • The MD5 checksum of the file: b01f166f87cdc9db3c0eda48053a46ab.
    • User Manual

Mobile Security Labs

  • Android Repackaging Lab

    Insert malicious code inside an existing Android app, and repackage it.

  • Android Device Rooting Lab

    Develop an OTA (Over-The-Air) package from scratch to root an Android device.
